
# Interactive GraphMath

Interactive GraphMath is an interactive math learning site for exploring equations, visualizing graphs, and understanding how formulas behave. It is designed to help students test ideas quickly, compare different functions, and see math concepts in a visual way.

## What the site does

- Lets users enter equations and immediately see them drawn on the graph.
- Supports both 2D and 3D graph views.
- Includes variables and graph settings so users can experiment with changes.
- Shows a bottom panel with graph information, built-in examples, and explanations.
- Uses AI-backed explanations when available, with a local fallback when the AI service is unavailable.
- Provides preset expressions to help users get started faster.

## Who it is for

This site is meant for students, teachers, and anyone learning algebra, trigonometry, calculus, or introductory 3D graphing. It can be used for class demonstrations, self-study, homework checking, and quick function exploration.

## How to run it

1. Install dependencies with `npm i`.
2. Start the frontend with `npm run dev`.
3. If you want AI explanations locally, start the API server with `npm run dev:api`.

## Notes

- The app is still a work in progress, so some features may change.
- The AI explanation feature depends on external API configuration.
